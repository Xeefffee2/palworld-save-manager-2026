# 📖 About

**Palworld Save Manager** is an open-source desktop application for Windows that provides a simple interface for browsing and managing local **Palworld** save files.

The application helps users access save file locations, organize local save data and create backup copies.

---

# ✨ Features

### 📂 Save Files

* Browse local save files
* View save folders
* Open save file locations
* View save file information

### 💾 Backup Copies

* Create backup copies
* Browse backup folders
* Restore backup copies

### 📄 Information

* File name
* Folder location
* File size
* Last modified date

### ⚙️ Application

* Desktop interface
* Dark mode
* Configurable application settings
